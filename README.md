# Diamond Price Prediction using SVR

This project focuses on predicting diamond prices using Support Vector Regression (SVR) based on various features such as carat, cut, color, and clarity.

## Project Overview

The goal is to build a regression model that can estimate diamond prices using machine learning techniques.

## Steps

- Data loading
- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training (SVR)
- Model evaluation

## Dataset

Dataset source: Kaggle
Dataset link: <https://www.kaggle.com/datasets/shivam2503/diamonds>

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
